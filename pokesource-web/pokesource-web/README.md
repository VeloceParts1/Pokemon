# PokeSource Web

PokeSource provides Pokémon Trading Card Game (TCG) enthusiasts with guaranteed authentic, direct-from-distributor sealed products.

## Tech Stack

- **Frontend:** React 19
- **Build Tool:** Vite 8
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Routing:** React Router DOM

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Directory Structure

- `src/components`: Reusable UI components (Navbar, Footer, etc.)
- `src/pages`: Application pages (Home, Store, Cart, FAQ)
- `src/context`: React context for state management (Cart, Auth)
- `src/assets`: Static assets (images, icons)
- `src/hooks`: Custom React hooks
- `src/utils`: Utility functions and constants

## Serving the Site

The site is served on port 3000.
